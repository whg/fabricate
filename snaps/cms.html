Flaunt

A CMS to showoff

January 2011

Flaunt is a server-side web application or CMS, which uses PHP to MySQL to store the content. It supports themes, which use a simple API to pull in content. All content is managed inside backend.

The idea for this came when I wanted to make a portfolio website for myself, but it was never made until I had to document a project for a University project.

  • The order of the sidebar items can be ordered in the browser by dragging and dropping boxes.
  • All .css and .js files are automatically included when they are in the correct directory.
  • The main stylesheet can be editing without any FTP/Terminal nonsense in the backend.
  • Loads of others...

The source can be downloaded/viewed on github, and I also made a profile on ohloh for it, where you can see an analysis.
