Image to ASCII

June 2011

During the summer of 2011 I wanted to learn Python and needed a small project, this was what I chose. Using a web interface you can submit an image, either on your filesystem or using a URL, and a ASCII representation is returned.

You can try it with your own images here.

The form is processed by a small PHP script, this saves the image on the server, and calls the Python script to create the ASCII file. When the script returns the PHP prints the contents of the file.

The returned text can either be formatted to be viewed in the browser or in a text document, where the line height is different.

Pedro & Sybil

A hardware/software marriage

May 2011

Pedro is a plotter and Sybil is the software that directs Pedro. This was my second year project at University.

The idea for the project spawned from the desire to write a beautifully written letter, a letter that would be written with a pen but have the precision only a machine can accomplish.

Pedro is built from basic materials, either custom made or recycled from printers and has the drawing area of A3 sized paper, it can hold a variety of different pen sizes. Connection to the computer is via USB and the electronics are built from basic components, using an Atmel ATmega8 as the micro-controller, which has been programmed in C.

Sybil is a native Mac OS X application written in C++ and Objective-C using openFrameworks. Users can add, preview and manipulate both text and image before sending the data to be plotted. Within Sybil, users can also issue commands to draw primitive shapes via a command line interface.

Full documentation including images, techniques and code can be seen at pedroandsybil.com
